# 🏫 Sistema de Gestão de Boletins - UniVap

Este projeto é uma solução completa para o gerenciamento acadêmico de notas, faltas e registros de alunos, desenvolvido sob o contexto das necessidades escolares do Colégio Univap. O sistema centraliza o controle pedagógico, permitindo uma visão clara do desempenho de cada estudante.

## 🚀 Funcionalidades

* **Gerenciamento de Alunos:** Cadastro, edição e consulta de informações dos estudantes.
* **Controle de Notas e Médias:** Sistema automatizado para lançamento de avaliações e cálculo de médias bimestrais.
* **Monitoramento de Frequência:** Registro detalhado de faltas para acompanhamento da assiduidade escolar.
* **Status Acadêmico:** Identificação automática de aprovação, retenção ou necessidade de recuperação.
* **Persistência de Dados:** Armazenamento seguro de todas as informações acadêmicas para consultas futuras.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** C#
* **Interface:** Windows Forms (WinForms)
* **Banco de Dados:** MySQL
* **Ambiente de Desenvolvimento:** Visual Studio
* **Conceitos:** Programação Orientada a Objetos (POO) e CRUD (Create, Read, Update, Delete).
